About

Version: -

Author: WebVideo2NAS Team

This extension allows you to send web videos (M3U8 streams, MP4 files) directly to your NAS for download.

How to use

  1. Configure NAS endpoint
  2. Browse video website
  3. URLs detected automatically
  4. Open extension icon
  5. Click "Send to NAS"

Troubleshooting

• Check NAS accessibility
• Verify API port (52052)
• Check API Key matches .env
• Check Firewall
